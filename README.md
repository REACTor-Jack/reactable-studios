# REACTable Studios

Marketing studio. Logo to souvenir. We build the whole campaign.

Live: https://reactable-studios.vercel.app

## Stack
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS 3
- Inter (display + sans), JetBrains Mono (accent)
- Vercel deploy

## Color system
- `rs-blue` `#00B0FF` — kinetic accent (REACT-inherited)
- `rs-orange` `#FF6B1A` — emerging contrast accent
- `rs-ink` `#0A0A0A` — type
- `rs-white` `#FFFFFF` — ground
- `rs-paper` `#FAFAFA` — secondary ground
- `rs-mute` `#6B6B6B` — supporting copy
